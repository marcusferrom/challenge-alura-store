Análise de Desempenho das Lojas - Alura Store

Este projeto analisa o desempenho de quatro lojas com o objetivo de
recomendar qual delas deve ser vendida. A análise foi realizada no
Google Colab utilizando Python, Pandas e Matplotlib.

Foram considerados os seguintes fatores: faturamento total, produtos e
categorias mais e menos vendidos, média de avaliações dos clientes e
frete médio por loja.

Os resultados mostraram que, embora a Loja 1 tenha o maior faturamento,
ela apresenta o maior frete médio e a menor média de avaliação. As
demais lojas demonstram maior equilíbrio entre receita, satisfação do
cliente e custo logístico.

Com base nesses indicadores, a recomendação foi a venda da Loja 1,
considerando o risco estratégico de longo prazo em comparação às demais
unidades.
